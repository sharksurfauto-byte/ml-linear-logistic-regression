# ML: Linear & Logistic Regression
**Author:** Aliasghar (Ali)  
**Tech stack:** Python, scikit-learn, pandas, matplotlib, Colab

## Overview
This repo contains two Google Colab notebooks demonstrating:
- Linear Regression: dataset, exploratory data analysis, model training, evaluation, and improvements.
- Logistic Regression: binary classification example with preprocessing, model, and metrics.

## How to view
Open the notebooks directly on GitHub or click the Colab links below:
- `notebooks/linear_regression.ipynb`
- `notebooks/logistic_regression.ipynb`

## Run locally
1. `python -m venv venv && source venv/bin/activate`  
2. `pip install -r requirements.txt`  
3. Open the notebooks with Jupyter or run via Colab.
